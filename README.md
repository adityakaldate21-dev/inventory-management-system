#  Inventory Management System

A responsive web-based Inventory Management System developed to manage stock, billing, and employee-related operations for a local bakery. This project focuses on simplifying daily business tasks and reducing manual effort through an intuitive interface.

---

##  Live Demo
🔗 https://adityakaldate21-dev.github.io/inventory-management-system/

---

##  Features

- 📦 Inventory Tracking (Add, Update, Delete Items)
- 💰 Billing System Interface
- 👨‍💼 Employee Management (basic UI)
- 📊 Organized dashboard layout
- 🎨 Responsive design (mobile + desktop)
- ⚡ Smooth animations and interactive UI
